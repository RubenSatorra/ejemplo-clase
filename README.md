*Ejemplo de uso de git en Despliegue de Aplicaciones Web*

Este es un repositorio creado desde un server debian en una máquina virtual para un trabajo de DAW. 
Se trata de una prueba de conexión a una base de datos de prueba MySQL (la de prueba "world"), llamándola desde un archivo PHP.
Este se ejecuta en un server Apache y muestra los datos (Ciudad, país y población). También se ha creado la documentación usando PHPDocs.

Es necesario por lo tanto tener un servidor Apache que esté sirviendo el proyecto y la base de datos descargada. En este caso,
se realizó utilizando PHPMyAdmin.

En el repositorio tenemos el archivo PHP llamado info.php, otro info2.php que sería una prueba de que estaba funcionando y
la carpeta DOCS en la que se generó la documentación con PHPDocs.
